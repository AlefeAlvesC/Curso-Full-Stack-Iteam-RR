# 🌐 Práticas – Fundamentos de Desenvolvimento Web (Módulo 03)

Este repositório contém as atividades práticas desenvolvidas durante o **Módulo 03 – Fundamentos de Desenvolvimento Web** do curso Full Stack. O objetivo é aplicar, de forma prática, os conceitos básicos de construção de páginas web utilizando **HTML, CSS e JavaScript**.

---

## 🎯 Objetivo

Consolidar os conhecimentos fundamentais do desenvolvimento web por meio da criação de páginas simples, responsivas e interativas, simulando cenários reais do dia a dia de um desenvolvedor.

---

## 📚 Conteúdos Praticados

### 🧱 HTML
- Estrutura básica de páginas web  
- Uso de tags semânticas  
- Criação de formulários  
- Inclusão de imagens, vídeos e áudios  
- Organização e boas práticas  

---

### 🎨 CSS
- Estilização de páginas  
- Seletores, propriedades e valores  
- Modelo de caixa (Box Model)  
- Layout com Flexbox e Grid  
- Responsividade  
- Design adaptativo  

---

### ⚡ JavaScript
- Variáveis e tipos de dados  
- Estruturas condicionais e loops  
- Manipulação do DOM  
- Eventos e interatividade  
- Validação de formulários  

---

## 💻 Atividades Desenvolvidas

- Criação de páginas web estáticas  
- Desenvolvimento de layouts responsivos  
- Construção de formulários interativos  
- Manipulação dinâmica de elementos HTML  
- Exercícios de lógica aplicados ao JavaScript  

---

## 🛠️ Ferramentas Utilizadas

- VS Code  
- Navegador Google Chrome  
- Chrome DevTools  
- Git e GitHub
